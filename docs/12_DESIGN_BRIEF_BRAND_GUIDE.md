# Design Brief / Brand and Style Guide

## Brand positioning

AgentPress is a practical AI content operations tool. It should feel like software for production work, not a toy writing assistant.

## Product voice

- Direct
- Calm
- Technical but readable
- Minimal hype
- Action-oriented

## UI personality

- Sparse
- Structured
- Professional
- Fast to scan
- Clear status and ownership

## Visual direction

- Light-first interface
- Muted neutral background
- Strong readable text
- Subtle borders
- Minimal accent color
- Compact panels
- Simple rows/cards

## Avoid

- Chatbot-first UX
- Neon AI styling
- Excessive gradients
- Mascots
- Complex dashboard chrome
- Decorative workflow canvases

## Possible tagline

Composable AI pipelines for WordPress content.

## Product copy guidelines

Use simple verbs:

- Add variable
- Add agent
- Run now
- Save
- Publish
- Pause
- Start
- Copy output

Avoid vague AI terms:

- Magic
- Supercharge
- Revolutionary
- Autonomous genius

## Imagery style

MVP needs little imagery. Generated thumbnails should be content-specific. Product UI should not rely on decorative art.

## Logo guidance

MVP can use a wordmark only:

```txt
AgentPress
```

Optional mark idea:

- stacked blocks
- small arrow path
- document + pipeline motif

Logo should work in one color.
